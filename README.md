# Reading

### 简介
 Reading: 一款基于WanAndroid OpenApi开发的Kotlin版本的符合Material Desgin的Android客户端。项目采用MVP+Kotlin开发，现主要功能包括：首页WanAndroid模块、工作台模块、我的模块。
